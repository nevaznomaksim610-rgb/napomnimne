# Бот напоминаний о мероприятиях и дедлайнах

Telegram-бот + email-агент. Бот показывает акселераторы, гранты и
амбассадорства с дедлайнами и напоминаниями. Агент сам пишет на почты
компаний, читает ответы, анализирует их через DeepSeek и обновляет статус.

Полное описание архитектуры — в [ARCHITECTURE.md](ARCHITECTURE.md).

## Быстрый старт

```powershell
# 1. Виртуальное окружение
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 2. Зависимости
pip install -r requirements.txt

# 3. Настройки
Copy-Item .env.example .env
#   → впиши BOT_TOKEN (минимум, чтобы запустить бота)

# 4. Демо-данные (необязательно, но удобно)
python seed_data.py

# 5. Запуск
python run.py
```

После запуска открой бота в Telegram и нажми `/start`.

## Структура

```
run.py            — точка входа (бот + планировщик)
config.py         — настройки из .env
seed_data.py      — демо-наполнение БД
app/db/           — модели, enum-ы, репозиторий
app/bot/          — хендлеры, клавиатуры, тексты
app/agent/        — email-клиент, DeepSeek-анализатор, pipeline, рассылка
app/scheduler/    — задачи по расписанию
```

## Что нужно от заказчика, чтобы довести до прода

| # | Что | Куда | Приоритет |
|---|-----|------|-----------|
| 1 | Токен Telegram-бота (@BotFather) | `.env: BOT_TOKEN` | 🔴 обязательно |
| 2 | Ключ DeepSeek API | `.env: DEEPSEEK_API_KEY` | 🟡 для агента |
| 3 | Почта агента (адрес, SMTP/IMAP, app-password) | `.env: EMAIL_*` | 🟡 для агента |
| 4 | База почт компаний (категория, название, email, дедлайн, ссылка, описание) | таблица/CSV | 🟢 контент |
| 5 | Тексты писем и кадэнс рассылки | `app/agent/outreach.py` | 🟡 для агента |
| 6 | Твой telegram_id для уведомлений агента | `.env: ADMIN_CHAT_ID` | 🟢 |
| 7 | Подтвердить набор категорий и этапов готовности | ARCHITECTURE.md §4–5 | 🟢 |

> База почт (п.4) — самый ожидаемый кусок. Пришли её в любом виде (таблица,
> список), и я сделаю импорт в БД.
